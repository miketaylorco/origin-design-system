# Colors

The Origin design system uses a three-tier color token architecture:

- **T1 Primitive** — the raw palette (50–950 scales per color family); exists in Figma only, not exposed as code tokens
- **T2 Semantic** — purpose-driven aliases with light/dark mode variants
- **T3 Component** — component-specific overrides (defined per component)

Toggle **Light / Dark** in the toolbar above to see semantic token values switch.
