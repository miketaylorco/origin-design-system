# Typography

Origin uses two type families with a responsive scale (Desktop / Tablet / Mobile).
Resize the viewport to see the scale adapt.

---

## Type scale — Heading

export const headingSizes = ["2xl", "xl", "lg", "md", "sm", "xs"];

      </p>
      <p style={{
        fontFamily: "var(--font-family-heading)",

        color: "var(--text-default)",
        margin: 0,
      }}>

      </p>

## Font families

  ))}
</div>
